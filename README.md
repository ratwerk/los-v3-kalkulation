# LOS V3 — Projektkalkulation

Monte-Carlo-Analyse für das Leadership Operating System V3.

Der Kunde kann sein Paket interaktiv konfigurieren — die Kostenverteilung aktualisiert sich in Echtzeit.

## Features

- Phasen ein-/ausblenden (mit Abhängigkeitsprüfung)
- Premium-Features toggle
- Standard vs. Split-Layout Auswahl (mit Vorschau-Link)
- 50.000 Monte-Carlo-Simulationen mit Dreiecksverteilung
- P10 / P50 / P90 Kostenanzeige
- Kostenaufteilung nach Phase

## Deploy

Vercel erkennt das Projekt automatisch als Static Site.
